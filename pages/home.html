CBT

Common Build Toolset (CBT) is an open source toolset for MSBuild.

Get Started »

Currently v2.0.0

Designed for all source code projects big and small.

CBT is great for small source code projects and scales to very large code bases. It is extremely light weight with a very low barrier of entry while its modules extend its core functionality to provide a rich and powerful feature set.

Light weight

CBT ships as just three files to include in your code base. Converting existing projects involves including one import.

Get started »

Extensible

CBT modules extend the core functionality to provide advanced MSBuild functionality such as versioning, signing, and much more.

Browse Modules »

Scalable

CBT can be used in the enterprise for very large projects and automated build systems such as Jenkins or Visual Studio Team System.
